<h1 align="center">LiveWallpaper</h1>

<p align="center">
  A lightweight, native macOS utility to set video files as your desktop wallpaper.
</p>

<p align="center">
  <code>Swift 5.9+</code> &nbsp;•&nbsp; <code>macOS 13.0+</code> &nbsp;•&nbsp; <code>MIT License</code>
</p>

---

## Features

- **Multi-Monitor Support** — Spans and scales video content across all connected displays.
- **Unconditional Looping** — Seamless, zero-gap video playback.
- **Menu Bar Control** — Pause/play, toggle audio, and adjust speed (0.25x to 3.0x) directly from the tray.
- **Gallery Manager** — Drag, drop, and manage wallpaper files in a sleek HUD interface.
- **Sandbox Safe** — Persists selected videos securely between app launches.

## Quick Start

Ensure you have Xcode Command Line Tools installed, then run:

```bash
make run
```

Other commands:
- `make` — Build the application bundle (`build/LiveWallpaper.app`).
- `make clean` — Remove build artifacts.
